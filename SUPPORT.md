# Support

## Public support

Use GitHub Issues for reproducible installation defects, runtime errors, compatibility problems, and documentation gaps. Include the PHP version, database product/version, web server, relevant logs with secrets removed, and exact reproduction steps.

## Private support

Use `support@vib.tools` for security disclosures or project-support questions that should not be public.

## Out of scope

The project cannot safely diagnose unknown production systems from complete database dumps, private keys, passwords, or customer data. Do not upload sensitive operational material.
